1. Project Overview
Signal Matrix is a multi-timeframe, probabilistic trading signal platform designed to identify high-conviction trade opportunities across a diversified universe of ~51 assets. The platform is built around the principle that simplicity and structural clarity outperform complex over-engineered systems. The methodology is rooted in fractal market theory, wave structure analysis, and probabilistic statistics rather than traditional lagging indicators.
Core Philosophy
Fewer, higher quality signals beat more, lower quality signals
Trend alignment across timeframes is the primary filter
Risk ranges are mathematically derived, not discretionary
The system tells you where the market is, not where you want it to be

C Update Logic — CRITICAL
C is NOT set once and frozen. C updates dynamically as the trend develops.
# UPTREND — C walks UP (higher lows)
new_pivot_low = find_most_recent_confirmed_pivot_low(prices, bar_window)
if new_pivot_low > current_C:
    current_C = new_pivot_low

# DOWNTREND — C walks DOWN (lower highs)
new_pivot_high = find_most_recent_confirmed_pivot_high(prices, bar_window)
if new_pivot_high < current_C:
    current_C = new_pivot_high

# Break check always uses CURRENT C — never stale C
if direction == UPTREND and current_price < current_C: BREAK_OF_TRADE
if direction == DOWNTREND and current_price > current_C: BREAK_OF_TRADE

ABC Pivot Search — All A Candidates Tried
For each (C, B) pair, all A candidates are iterated newest-first. The engine stops at the first A satisfying C > A (uptrend) or C < A (downtrend). The nearest A before B is not assumed to be correct.

2.3 Signal Components
1. Fractal Dimension (D)
D → 1.0 = trending, D → 1.5 = choppy, D → 2.0 = mean-reverting. Derived from Hurst: D = 2 − H.
2. Hurst Exponent (H) — DFA Method
H > 0.5 = trending, H < 0.5 = mean-reverting, H = 0.5 = random walk. Calculated at Trade (63d), Trend (252d), and Long Term (756d) timeframes independently.
3. Relative Implied Volatility (Rel IV%)
IV expressed as percentile of its own 252-day history. PRIMARY role: scales LRR/HRR width. NOT in the conviction score formula. Source: Schwab options chain (falls back to Yahoo proxy on token expiry).
4. OBV Volume Signal
Applied as multiplier to conviction score. OBV direction determined by pivot structure (bar_window = 9) applied to OBV series. Compared against Trade Dir (not Viewpoint).
5. VIX Macro Regime
Dashboard header indicator — not a per-ticker signal. VIX < 20 = Green (Investable), 20–29 = Amber (Choppy), ≥ 30 = Red (Danger).

2.7 LRR / HRR Formula — v1.6 (Redesigned)
Two bugs fixed from v1.5 formula: (1) H_factor was inverted — high H now correctly means shallow pullback. (2) Anchor switches from C to B when price has extended well beyond B.
H_factor Lookup

Uptrend Formulas
bc_range = abs(b - c)
anchor   = c if d < b + bc_range else b   # switch to B when extended
lrr = anchor + bc_range * (1 - hf) - sigma * lrr_mult
hrr = d      + bc_range * hf        + sigma * hrr_mult
Downtrend Formulas (Mirror)
bc_range = abs(b - c)
anchor   = c if d > b - bc_range else b   # switch to B when extended
hrr = anchor - bc_range * (1 - hf) + sigma * hrr_mult
lrr = d      - bc_range * hf        - sigma * lrr_mult
Rel IV Multipliers (sigma scaling)

2.9 ENTRY Signal Logic (v1.6 — NEW)
New column in the dashboard table between Conviction % and Trade Dir. Reads from existing signal data — no new backend endpoint.
BUY Conditions (all four must be true)
Viewpoint = Bullish
Trade Dir = Bullish
Trend Dir = Bullish
Math.abs(close - tradeLRR) / tradeLRR <= 0.02 (price within 2% of Trade LRR)
SELL Conditions (all four must be true)
Viewpoint = Bearish
Trade Dir = Bearish
Trend Dir = Bearish
Math.abs(close - tradeHRR) / tradeHRR <= 0.02 (price within 2% of Trade HRR)
Neutral viewpoint never triggers ENTRY signal regardless of price proximity. ENTRY count displayed in header summary row alongside BULLISH / BEARISH / ALIGNED / ALERTS.
2.10 OBV Pivot Engine
Replaces the price-momentum proxy entirely. OBV direction determined by applying the same pivot structure logic as the price engine (bar_window = 9) to the OBV series. Vol Signal is compared against Trade Dir (not Viewpoint) — volume is a short-term signal confirming the trade timeframe move.

2.11 Alert Flag Trigger Conditions
All three must be simultaneously true:
Trade H > 0.55 AND Trend H > 0.55
Viewpoint = Bullish OR Bearish (never fires on Neutral)
Final Conviction >= 70% (after OBV volume multiplier)
